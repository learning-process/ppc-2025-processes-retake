## 1. Введение

Нахождение минимальных значений по столбцам матрицы — распространённая операция в статистике, обработке данных и линейной алгебре. При работе с большими матрицами параллельное вычисление позволяет существенно сократить время. В данной работе реализован параллельный алгоритм поиска минимумов по столбцам с использованием MPI.

## 2. Постановка задачи

**Входные данные:** матрица целых чисел размером rows × cols, представленная как одномерный вектор.

**Выходные данные:** вектор минимальных значений длиной cols, где `result[j]` — минимальный элемент j-го столбца.

**Ограничения:** матрица непустая, rows > 0, cols > 0.

## 3. Базовый алгоритм (последовательный)

Для каждого столбца j от 0 до cols-1:
1. Инициализировать минимум значением `INT_MAX`.
2. Пройти по всем строкам i и обновить минимум: `min = std::min(min, matrix[i * cols + j])`.
3. Записать результат в `result[j]`.

Сложность: O(rows × cols).

## 4. Схема распараллеливания

Строки матрицы распределяются между процессами равномерно:
- Каждый процесс получает свой диапазон строк.
- Каждый процесс вычисляет локальные минимумы по столбцам для своих строк.
- Глобальные минимумы собираются на root-процессе через `MPI_Reduce` с операцией `MPI_MIN`.

**Коммуникации:**
1. Root рассылает данные матрицы через `MPI_Scatter`.
2. Каждый процесс вычисляет локальные минимумы.
3. `MPI_Reduce` с `MPI_MIN` собирает глобальные минимумы на root.
